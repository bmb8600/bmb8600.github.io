Project Purpose and Overview

This website was created for my Intro to Web Development (IT 3203) course to explore and explain the concept of Progressive Web Applications (PWAs). The goal of this project is to provide a comprehensive overview of PWAs and their features while showcasing and applying web development skills.


This site consists of several pages that cover different topics related to PWAs:

  • Home Page: An introduction to PWAs and their key features.
  • Key Concepts: A table of definitions of the key concepts discussed.
  • How PWAs Work: An in-depth look at the technologies behind PWAs, including service workers, web app manifests, and more.
  • PWAs vs. Apps and Websites: A comparison between PWAs, native apps, and traditional websites, highlighting their differences and advantages.
  • References: A list of references used in the creation of this project.
  • About Page: The current page, where I introduce myself and provide information about the project.
